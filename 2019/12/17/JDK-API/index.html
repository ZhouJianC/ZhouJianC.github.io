2. String类

String类不可变。因为用于存放字符的数组被声明为final，因此只能赋值一次，不能修改。所以一旦创建了String类对象，它的值无法修改。如果需要对字符串做修改，可选择StringBuffer&StringBuilder类

字符串操作的常用方法：
int length()
int indexOf(int ch)
int indexOf(String str)
int lastIndexOf(int ch)
int lastIndexOf(String str)
String substring(int beginIndex)
String sunstring(int beginIndex, int endIndex)
String trim()
String toLowerCase()
String toUpperCase()
String[] split(String regex, int limit)
String copyValueOf(char[] data)
String copyValueOf(char[] data, int offset, int count)
String valueOf(primitive data type x)
返回data type类型x参数的字符串表示形式
boolean equals(Objects obj)
boolean startsWith(String prefix)
boolean startsWith(String prefix, int toffset)
boolean endsWith(String suffix)
char charAt(int index)
char[] toCharArray()
byte[] getBytes()
